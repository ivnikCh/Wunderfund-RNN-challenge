# Wunderfund-RNN-challenge

В 'pdf' расположено краткое объяснение основной используемой модели (объяснены ключевые моменты построения линейной модели, за исключением назначительных моментов работы с фичами

Dataresearch.ipynb - малая часть проведенного ресерча данных

train.py - файл с пайплайном обучения модельки (отвалидировал с помощью llm-ки чтобы были комменты и прослеживалась структура по названиям)

solution.py и папка с весами  (сам класс модельки, тоже провалидированный для читабельности кода как это просили на соревновании и utils.py - скрипт для прогона)
